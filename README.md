## 참여 방법
1. 이 저장소를 fork 한다.
2. 본인 GitHub에 생성된 원격 저장소로 간다. 
3. 해당 주차 > 해당 일자(문제) 폴더로 가서 본인 이름으로 된 파일을 생성한다.
   + 파일명 : `이름.md`
4. 양식에 맞게 풀이를 작성한다.
5. 원본 저장소로 Pull Request를 한다.    
<br/>
  
+ 상세 설명 참고 : https://waytocse.tistory.com/59

## 마크다운 양식

+ 풀이 개수 제한 X (3개 이상이면 밑에 계속 써주세요)

```
#### 풀이 1. 풀이 제목

풀이 설명 & 과정 (또는 코드 중간에 주석 작성)


코드 입력하기


실행 시간 : 000ms    
사용 공간 : 0000KB  
풀이 시간 : 00분  

--- 

#### 새롭게 알게된 점
  + (있으면 작성)

#### 어려웠던 점
  + (있으면 작성)
  
```
