## 참여 방법
1. 이 저장소를 fork 한다.
2. 본인 GitHub에 생성된 원격 저장소로 간다. 
3. 해당 주차 > 해당 일자(문제) 폴더로 가서 본인 이름으로 된 파일을 생성한다.
   + 파일명 : `이름.md`
4. 양식에 맞게 풀이를 작성한다.
5. 원본 저장소로 Pull Request를 한다.   
6. 다른 사람 PR에 자유롭게 코드 리뷰를 한다.
7. 리뷰가 완료되면 Merge 하겠습니다. 
<br/>
  
+ 상세 설명 참고 : https://waytocse.tistory.com/59
